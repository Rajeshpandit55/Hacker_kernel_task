# Product App

A simple Project with Frontend and backend Setup with Authentaction and Authorization and user dashboard.

## Getting start

### To start Frontend 
1. Navigate to the `frontend` directory:
     ```bash
     cd frontend
     ```
     ```bash
     npm install
     ``` bash
     npm run start
     ```
### to start Backend
2.Navigate to the `backend` directory:
  ```bash
    cd backend
  ```
```bash
npm install
```
``bash
npm run dev
```
